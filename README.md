# 个人记账管理系统 python350

## 项目概述

个人记账管理系统是基于 Python Django 框架开发的一个便捷的财务记录工具。系统旨在帮助用户高效地管理个人财务，包括登录验证、个人信息管理、收入支出记录以及相关的统计功能。

## 技术栈

- **后端**: Python Django
- **数据库**: MySQL
- **前端**: HTML

## 功能模块

### 用户模块

- 用户登录
- 个人信息修改

### 财务管理模块

- 收入记录
- 支出记录
- 收入记账管理
- 支出记账管理

### 统计模块

- 收入统计
- 支出统计

## 系统角色

- 普通用户：负责日常的财务记录和管理

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x

## 部署步骤

1. 安装 Python 和 Django
2. 配置 MySQL 数据库
3. 迁移数据库
4. 运行 Django 应用
